Agentic Chatbot Setup and Usage

Initial Setup

  1. Create Conda Environment 
    conda create -n agent-bot-main-env python==3.12 -y
  2. Activate the Created Environment 
    conda activate agent-bot-main-env
  3. Install Required Dependencies 
    pip install -r requirements.txt

Running the Application

Step 1: Activate the Environment 

conda activate agent-bot-main-env

Step 2: Start the Agentic Chatbot

Run the main_agentic_chatbot.py file: 

python main_agentic_chatbot.py

Step 3: Test the Agent

Run the main_test_agent.py file: 

python main_test_agent.py

Step 4: Open the Frontend

Open the index.html file in a web browser to interact with the chatbot.

OR

Alternative Method

You can run the application using the provided batch or shell scripts:

  • On Windows: Run application_agentic_bot.bat
  • On macOS/Linux: Run application_agentic_bot.sh
